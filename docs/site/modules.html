模块生态

覆盖教学、创客、竞赛的模块库

PicoStudio 提供官方与社区维护的驱动、示例、课程与 AI 生成模块，让 RP2040 / RP2350 项目快速落地。

模块概览 
• 官方模块：传感器、显示、音频、无线
• AI 生成：按需求定制项目脚手架
• 教学资源：课程包、实验指导、课后作业
• 社区精选：竞赛作品、创客项目、插件

官方模块

🌡️

传感器套件

DHT、BME280、光照、IMU、超声波测距，提供统一 API 与调试仪表板。

🖼️

显示与交互

支持 LCD、OLED、E-Ink 与电容触控，内置 UI 组件库与布局模板。

🔊

音频与多媒体

提供音效播放器、MIDI 接口、语音合成与声波可视化工具。

📡

无线通信

Wi-Fi、蓝牙、LoRa 与 Zigbee 适配，支持联机调试与 OTA 更新。
